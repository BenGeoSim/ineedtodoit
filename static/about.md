# About ineedtodo.it

Welcome to **ineedtodo.it** — a *minimalish* todo app with just the features that matter.

---

## Features

### Tasks
- **Infinite nesting** — break any task into subtasks, and subtasks into subtasks, as deep as you need
- **Priorities** — 5 levels (urgent → none), shown as a coloured dot; click to cycle through
- **Due dates** — set a deadline on any task; shown in the task's priority colour when less than 7 days away
- **Descriptions / notes** — attach a full markdown-friendly note to any task; click the **note** pill to view it
- **Tags** — add multiple tags per task for filtering; hide all tag badges with the tag toggle button
- **Subtask toggle** — collapse and expand nested children with the chevron button

### Organisation
- **Sort** — order your list by creation date or priority
- **Filter by tag** — click any tag badge in the filter bar to show only matching tasks
- **Filter by priority** — filter to only show tasks of a specific priority level
- **Move to space** — relocate a task (and all its subtasks) to a different space from the task menu

### Spaces
- **Personal space** — your private default list, only visible to you
- **Shared spaces** — create a space and invite others by email; all members see and edit the same list in real time
- **Add members** — grow a shared space beyond two people at any time
- **Rename spaces** — give your shared spaces meaningful names
- **Creator badge** — in shared spaces, a badge shows who created each task

### Account
- **Google sign-in** — one-click login with your Google account
- **Email & password** — create an account with any email address and password
- **Sync across devices** — your tasks follow you wherever you log in

### Data
- **Export to XML** — download all your tasks as an XML file at any time
- **Trash & restore** — deleted tasks go to the trash; restore them if you change your mind

## Tips for less obvious features

| Feature | How to use it |
|---|---|
| Add a subtask | Click the **↩** button to the left of the priority dot, or use the task menu |
| Set a due date | Open the task menu (⋮) and click the calendar icon |
| Attach a note | Open the task menu (⋮) and click the document icon |
| View an attached note | Click the **note** pill on the task |
| Hide tag badges | Click the eye/tag button to the left of the **All tags** filter |
| Move a task | Open the task menu (⋮) and click the arrow icon |
| Restore a deleted task | Switch to **View Trash** from the menu, then click the restore icon |
